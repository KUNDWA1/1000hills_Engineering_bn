User roles

One system, three powerful user experiences

The marketplace supports Customer, Vendor, and Admin dashboards, each optimized for the right workflow.

Customer / User

Browse products, request quotations, place orders, make payments, and track delivery, without choosing vendors directly.

  • Create an account and login
  • Browse and filter products
  • Add to cart and pay securely
  • Track order status and invoices
Vendor / Seller

Verified vendors add and manage products, set availability, and receive assigned orders from the admin.

  • Complete profile and upload verification
  • Publish products with stock and pricing
  • Update availability status
  • Track assigned orders and deliveries
Admin

The admin manages vendors, approves registrations, assigns orders, monitors payments, and reviews analytics.

  • Review vendor documents and approve access
  • Assign best vendor by price, reliability, stock and availability
  • Monitor payments and invoices
  • Track performance and generate reports

System workflow

How orders move through 1000Hills Engineering

Customer places an order or quotation request
Admin receives request
System shows matching vendors
Admin compares price, stock, reliability, availability
Admin selects the best vendor
Vendor receives the assigned order
Vendor processes the order and updates delivery
Customer receives product
